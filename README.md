# Анализ данных о детях-сиротах в РФ

Разведочный анализ (EDA) статистики по детям, оставшимся без попечения родителей, в Российской Федерации. Работа выполнена в ноутбуке [`sirotki.ipynb`](sirotki.ipynb).

## О проекте

В ноутбуке исследуется динамика показателей по детям-сиротам за ряд лет и в разрезе регионов. Среди рассматриваемых сюжетов:

- численность детей-сирот, выявленных и учтённых на конец года;
- количество детей, принятых на воспитание в семьи, и отменённые решения о передаче в семью;
- коэффициент возврата детей из семей (по регионам и годам);
- численность специалистов по охране детства и организаций для детей-сирот;
- региональные различия и выбросы (например, аномально высокие показатели в отдельных субъектах).

Отдельное внимание уделяется всплеску показателей в 2014–2015 годах и его интерпретации (экономическая ситуация, курс рубля, ФЗ № 272-ФЗ о запрете усыновления сирот иностранцами, гео-политические факторы).

## Данные

Ноутбук использует CSV-файл `data_orphans_127_v20241218.csv` (разделитель `;`), который должен лежать рядом с ноутбуком в корне проекта. Файл большой (~520 МБ) и в репозиторий не включён.

Ключевые поля датасета:

- `region`, `okato` — регион и его код ОКАТО;
- `year` — отчётный год;
- `indicator`, `indicator_category`, `indicator_value` — показатель, его категория и значение;
- `dimension` — измерение/разрез показателя.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

1. Поместите файл `data_orphans_127_v20241218.csv` в корень проекта.
2. Запустите Jupyter и откройте ноутбук:

```bash
jupyter lab sirotki.ipynb
```

3. Выполните ячейки последовательно (Run All).
